## 6. 관련 개념 매핑

### 6.1 shokasonjuku vs Laws of UX 대응

| shokasonjuku 개념 | Laws of UX 대응 |
|------------------|-----------------|
| 인지 부하 | Cognitive Load |
| 도허티 임계값 | Doherty Threshold |
| 피크-엔드 법칙 | Peak-End Rule |
| 목표 기울기 효과 | Goal-Gradient Effect |
| 자이가르닉 효과 | Zeigarnik Effect |
| 계열 위치 효과 | Serial Position Effect |
| 미적 사용성 효과 | Aesthetic-Usability Effect |
| 기본값 편향 | (Status Quo Bias) |
| 시각적 계층 | (Gestalt Principles) |
| 선택적 주의 | Selective Attention |

### 6.2 카테고리별 분류

```
UX 심리학 개념
├── 인지 (Cognition)
│   ├── 인지 부하
│   ├── 작업 기억
│   ├── 밀러의 법칙
│   └── 청킹
├── 지각 (Perception)
│   ├── 게슈탈트 원칙
│   ├── 시각적 계층
│   └── 선택적 주의
├── 의사결정 (Decision Making)
│   ├── 힉의 법칙
│   ├── 앵커 효과
│   ├── 미끼 효과
│   └── 기본값 편향
├── 동기 (Motivation)
│   ├── 손실 회피
│   ├── 희소성
│   ├── 게이미피케이션
│   └── 목표 기울기
├── 기억 (Memory)
│   ├── 자이가르닉 효과
│   ├── 계열 위치 효과
│   └── 피크-엔드 법칙
└── 신뢰 (Trust)
    ├── 사회적 증명
    ├── 후광 효과
    └── 미적 사용성 효과
```
