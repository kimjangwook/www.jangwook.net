## 핵심 발견사항

### 1. UX 심리학의 주요 카테고리

| 카테고리 | 핵심 개념 |
|----------|----------|
| 인지 부하 | 인지 부하, 밀러의 법칙, 청킹 |
| 행동 유도 | 넛지, 기본값 편향, 손실 회피 |
| 시각적 인식 | 시각적 계층, 게슈탈트 원칙 |
| 사용자 기억 | 계열 위치 효과, 자이가르닉 효과 |
| 동기 부여 | 게이미피케이션, 변동형 보상 |
| 신뢰 구축 | 사회적 증명, 후광 효과 |

### 2. 프론트엔드 개발 시 우선 적용할 개념

1. **도허티 임계값 (0.4초)** - 응답 시간 최적화
2. **인지 부하 감소** - 정보 청킹, 단계적 공개
3. **시각적 계층** - 명확한 정보 우선순위
4. **사회적 증명** - 신뢰 요소 표시
5. **미적 사용성 효과** - 세련된 디자인
