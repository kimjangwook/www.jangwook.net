### 개인정보처리방침

**최종 수정일: 2025년 11월 19일**

#### 1. 개요
jangwook.net(이하 "사이트")은 방문자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보처리방침은 사이트가 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

#### 2. 수집하는 정보

##### 2.1 자동으로 수집되는 정보
- IP 주소
- 브라우저 유형 및 버전
- 운영 체제
- 방문 페이지 및 체류 시간
- 참조 URL

##### 2.2 쿠키 및 유사 기술
본 사이트는 다음 목적으로 쿠키를 사용합니다:
- 사용자 경험 향상
- 웹사이트 트래픽 분석
- 맞춤형 광고 제공

#### 3. 정보 사용 목적
수집된 정보는 다음 목적으로 사용됩니다:
- 웹사이트 운영 및 개선
- 트래픽 분석 및 사용자 행동 이해
- 관련성 있는 콘텐츠 및 광고 제공

#### 4. 제3자 서비스

##### 4.1 Google Analytics
본 사이트는 Google Analytics를 사용하여 웹사이트 트래픽을 분석합니다. Google Analytics는 쿠키를 사용하여 익명의 사용 데이터를 수집합니다.
- [Google Analytics 개인정보처리방침](https://policies.google.com/privacy)
- [Google Analytics 옵트아웃](https://tools.google.com/dlpage/gaoptout)

##### 4.2 Google AdSense
본 사이트는 Google AdSense를 사용하여 광고를 표시합니다. Google AdSense는 쿠키를 사용하여 맞춤 광고를 제공합니다.
- [Google 광고 개인정보처리방침](https://policies.google.com/technologies/ads)
- [광고 개인 최적화 설정](https://www.google.com/settings/ads)

#### 5. 데이터 보관
수집된 데이터는 필요한 기간 동안만 보관되며, 더 이상 필요하지 않을 경우 안전하게 삭제됩니다.

#### 6. 사용자 권리
귀하는 다음과 같은 권리를 가집니다:
- 수집된 데이터 열람 요청
- 데이터 수정 또는 삭제 요청
- 데이터 처리 제한 요청
- 쿠키 사용 거부 (브라우저 설정을 통해)

#### 7. 어린이 개인정보
본 사이트는 만 14세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다.

#### 8. 개인정보처리방침 변경
본 방침은 변경될 수 있으며, 변경 시 이 페이지에 업데이트됩니다. 중요한 변경 사항이 있을 경우 사이트를 통해 공지합니다.

## Astro 컴포넌트 구현 예시

```astro
---
// src/pages/[lang]/privacy.astro
import BaseHead from '../../components/BaseHead.astro';
import Header from '../../components/Header.astro';
import Footer from '../../components/Footer.astro';
import { SITE_TITLE } from '../../consts';

export function getStaticPaths() {
  return [
    { params: { lang: 'ko' } },
    { params: { lang: 'en' } },
    { params: { lang: 'ja' } },
    { params: { lang: 'zh' } },
  ];
}

const { lang } = Astro.params;

const titles = {
  ko: '개인정보처리방침',
  en: 'Privacy Policy',
  ja: 'プライバシーポリシー',
  zh: '隐私政策',
};

const title = titles[lang as keyof typeof titles];
---

<!doctype html>
<html lang={lang}>
  <head>
    <BaseHead title={`${title} | ${SITE_TITLE}`} description={title} />
  </head>
  <body>
    <Header />
    <main>
      <article>
        <h1>{title}</h1>
        <!-- 언어별 콘텐츠 렌더링 -->
      </article>
    </main>
    <Footer />
  </body>
</html>
```
