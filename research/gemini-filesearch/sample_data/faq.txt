자주 묻는 질문 (FAQ)
===================

일반 질문
--------

Q1: SuperWidget Pro는 무엇인가요?
A: SuperWidget Pro는 개인과 팀의 생산성을 향상시키는 통합 작업 관리 도구입니다.
   프로젝트 관리, 자동화, 협업 기능을 하나의 플랫폼에서 제공합니다.

Q2: 무료 버전이 있나요?
A: 네, 14일 무료 체험판을 제공합니다. 체험 기간 동안 모든 프리미엄 기능을 사용할 수 있습니다.
   체험 후에는 무료 플랜(기본 기능)과 프리미엄 플랜 중 선택할 수 있습니다.

Q3: 여러 기기에서 사용할 수 있나요?
A: 네, 동일한 계정으로 최대 5대의 기기에서 동시 사용이 가능합니다.
   모든 데이터는 클라우드에 자동으로 동기화됩니다.

Q4: 오프라인에서도 사용할 수 있나요?
A: 네, 오프라인 모드를 지원합니다. 인터넷 연결이 복원되면 자동으로 동기화됩니다.


구매 및 결제
-----------

Q5: 가격은 어떻게 되나요?
A: 플랜별 가격:
   - 무료 플랜: $0/월 (기본 기능)
   - 프로 플랜: $9.99/월 (개인 사용자)
   - 팀 플랜: $19.99/월/사용자 (5명 이상)
   - 기업 플랜: 별도 문의

Q6: 환불이 가능한가요?
A: 구매 후 30일 이내에 100% 환불이 가능합니다.
   환불 요청: support@superwidget.com로 이메일 전송

Q7: 결제 방법은 무엇이 있나요?
A: 신용카드 (Visa, MasterCard, AMEX), PayPal, 계좌이체를 지원합니다.

Q8: 구독을 취소하면 어떻게 되나요?
A: 구독 기간 종료 시까지는 모든 기능을 계속 사용할 수 있습니다.
   이후에는 무료 플랜으로 자동 전환됩니다.


설치 및 설정
-----------

Q9: 지원하는 운영체제는 무엇인가요?
A: Windows 10 이상, macOS 12 이상, Linux (Ubuntu 20.04 이상)를 지원합니다.
   모바일 앱: iOS 14+, Android 10+

Q10: 설치가 실패합니다. 어떻게 해야 하나요?
A: 다음을 확인해주세요:
    - 관리자 권한으로 실행
    - 바이러스 백신 프로그램 일시 중지
    - 충분한 디스크 공간 확보 (최소 500MB)
    - 최신 버전 다운로드 확인

Q11: 초기 설정은 어떻게 하나요?
A: 첫 실행 시 설정 마법사가 자동으로 실행됩니다.
   1. 계정 생성 또는 로그인
   2. 작업 공간 설정
   3. 동기화 옵션 선택
   4. 알림 설정


기능 및 사용법
-------------

Q12: 팀원을 초대하려면 어떻게 하나요?
A: Settings > Team > Invite Members에서 이메일 주소를 입력하세요.
   초대된 사용자는 이메일을 통해 가입 링크를 받게 됩니다.

Q13: 프로젝트를 공유할 수 있나요?
A: 네, 프로젝트 우측 상단의 "Share" 버튼을 클릭하고,
   공유 링크를 생성하거나 특정 사용자를 초대할 수 있습니다.

Q14: 데이터를 내보내기(export)할 수 있나요?
A: File > Export에서 다양한 형식으로 내보내기가 가능합니다:
   - CSV, Excel, JSON, PDF

Q15: 자동화 기능은 어떻게 설정하나요?
A: Tools > Automation > New Rule에서:
   1. 트리거 조건 설정 (예: 작업 완료 시)
   2. 실행할 작업 선택 (예: 이메일 전송)
   3. 규칙 저장 및 활성화


기술 지원
--------

Q16: 비밀번호를 잊어버렸습니다.
A: 로그인 페이지에서 "Forgot Password?"를 클릭하세요.
   등록된 이메일로 재설정 링크가 전송됩니다.

Q17: 앱이 느려졌습니다. 어떻게 해야 하나요?
A: 다음을 시도해보세요:
   - 캐시 삭제: Settings > Advanced > Clear Cache
   - 플러그인 비활성화: Tools > Plugins > Disable All
   - 앱 재시작
   - 최신 버전으로 업데이트

Q18: 데이터를 복구할 수 있나요?
A: 모든 데이터는 30일 동안 백업됩니다.
   Settings > Data > Restore Backup에서 복구할 수 있습니다.

Q19: 기술 지원은 어떻게 받나요?
A: 여러 채널을 통해 지원을 받을 수 있습니다:
   - 이메일: support@superwidget.com (24시간 내 응답)
   - 라이브 채팅: 웹사이트에서 가능 (평일 9:00-18:00 KST)
   - 전화: 1588-1234 (평일 9:00-18:00 KST)
   - 커뮤니티 포럼: community.superwidget.com

Q20: 버그를 발견했습니다.
A: bugs@superwidget.com으로 다음 정보를 보내주세요:
   - 운영체제 및 앱 버전
   - 버그 재현 방법
   - 스크린샷 또는 오류 로그


보안 및 개인정보
--------------

Q21: 데이터는 안전한가요?
A: 네, 다음 보안 조치를 적용하고 있습니다:
   - 256비트 AES 암호화
   - SSL/TLS 전송 암호화
   - ISO 27001 인증 데이터센터
   - 정기적인 보안 감사

Q22: 데이터는 어디에 저장되나요?
A: AWS 클라우드 (서울 리전)에 저장됩니다.
   기업 플랜의 경우 온프레미스 배포도 가능합니다.

Q23: 계정을 삭제하면 데이터는 어떻게 되나요?
A: 계정 삭제 후 30일 이내에는 복구 가능합니다.
   30일 이후에는 모든 데이터가 영구 삭제됩니다.


업데이트 및 버전
--------------

Q24: 최신 버전은 어디서 다운로드하나요?
A: 공식 웹사이트(www.superwidget.com/download)에서 다운로드할 수 있습니다.
   앱 내에서도 Help > Check for Updates로 업데이트 가능합니다.

Q25: 자동 업데이트를 끌 수 있나요?
A: Settings > Updates > Automatic Updates에서 끄고 켤 수 있습니다.
   하지만 보안 및 안정성을 위해 자동 업데이트를 권장합니다.

Q26: 구 버전으로 되돌릴 수 있나요?
A: 문제가 있는 경우에 한해 이전 버전으로 되돌릴 수 있습니다.
   support@superwidget.com으로 문의해주세요.


기타
----

Q27: 기업용 플랜의 혜택은 무엇인가요?
A: 기업 플랜은 다음을 포함합니다:
   - 무제한 사용자
   - 전담 계정 매니저
   - 우선 기술 지원
   - SSO (Single Sign-On) 지원
   - 온프레미스 배포 옵션
   - 맞춤형 교육 프로그램

Q28: 교육 기관 할인이 있나요?
A: 네, 학생 및 교육 기관은 40% 할인을 받을 수 있습니다.
   education@superwidget.com으로 학교 이메일로 문의해주세요.

Q29: API 문서는 어디서 볼 수 있나요?
A: https://docs.superwidget.com/api에서 전체 API 문서를 확인할 수 있습니다.

Q30: 더 많은 도움말은 어디서 찾을 수 있나요?
A: 다음 리소스를 활용하세요:
   - 온라인 도움말: help.superwidget.com
   - 비디오 튜토리얼: youtube.com/superwidget
   - 블로그: blog.superwidget.com
   - 커뮤니티: community.superwidget.com
