# 웹 퍼블리셔를 위한 E2E 페이지 테스트 자동화 시스템

## 개요

웹 퍼블리셔가 페이지 개발 후 브라우저 테스트를 수행할 때 필요한 모든 검증 항목을 하나의 커맨드로 자동화하는 시스템입니다. Claude Code와 Playwright를 활용하여 품질 보증 프로세스를 표준화하고 자동화합니다.

## 목적

1. **품질 보증 표준화**: 모든 페이지가 동일한 기준으로 검증되도록 테스트 항목 표준화
2. **효율성 향상**: 수동 테스트 시간을 80% 이상 단축
3. **일관성 확보**: 사람의 실수나 누락 없이 모든 항목을 자동으로 검증
4. **빠른 피드백**: 개발 직후 즉각적인 품질 검증 결과 제공
5. **지속적 개선**: 테스트 결과를 바탕으로 반복적인 문제 해결

## 주요 기능

### 1. 크로스 브라우저 테스트
- Chromium, Firefox, WebKit (Safari) 동시 테스트
- 브라우저별 렌더링 차이 자동 검출
- 스크린샷 비교를 통한 시각적 회귀 테스트

### 2. 링크 무결성 검증
- 페이지 내 모든 링크 크롤링
- 깨진 링크 (404, 500 등) 자동 감지
- 외부 링크 접근성 확인
- 상대 경로 vs 절대 경로 검증

### 3. UI/UX 검증
- 레이아웃 브레이크 포인트 테스트
- 모바일 반응형 디자인 검증
- 터치 타겟 크기 확인 (최소 44x44px)
- 폰트 크기 가독성 검증

### 4. 콘텐츠 품질 검증
- 오탈자 자동 검출 (맞춤법 검사)
- 텍스트 가독성 분석
- 메타데이터 완성도 체크

### 5. 인터랙션 테스트
- 버튼 클릭 이벤트 발화 검증
- 호버 이벤트 동작 확인
- 폼 입력 및 검증 테스트
- 애니메이션 성능 체크

### 6. 이미지 최적화 검증
- 렌더링 사이즈 vs 내추럴 사이즈 비교
- 과도하게 큰 이미지 자동 감지
- WebP/AVIF 포맷 사용 권장
- 이미지 지연 로딩 확인
- Alt 텍스트 누락 체크

### 7. 접근성 (a11y) 검증
- WCAG 2.1 AA 레벨 준수 확인
- 키보드 네비게이션 테스트
- 스크린 리더 호환성
- 색상 대비 비율 검증
- ARIA 속성 검증

### 8. SEO 검증
- 메타 태그 완성도 체크
- Open Graph 데이터 검증
- 구조화된 데이터 (Schema.org) 확인
- 페이지 속도 측정
- Core Web Vitals 검증

## 기술 스택

- **Claude Code**: 워크플로우 오케스트레이션, 결과 분석, 리포트 생성
- **Playwright**: 브라우저 자동화 및 E2E 테스트
- **Axe-core**: 접근성 검증
- **Lighthouse**: 성능 및 SEO 측정
- **TypeScript**: 테스트 코드 작성

