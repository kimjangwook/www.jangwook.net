## 著者紹介

開発者としてAIツールを活用した生産性向上に深い関心を持っています。実際のブログ運営プロジェクトで11個のAIエージェントシステムを構築・運営した経験を基に、理論ではなく実践で検証された知識を共有します。

71個のブログ記事を執筆しながら蓄積したノウハウと試行錯誤をこの本に込めました。

---

## この本の特徴

### 1. Cookbookスタイル
この本は最初から最後まで順序通りに読む必要はありません。必要なレシピを見つけてすぐに適用できるように設計されています。

### 2. 実践検証されたコード
すべての例は実際のプロジェクトで使用中のコードです。理論的な例示ではなく、すぐにコピーして使用できる実用的なコードを提供します。

### 3. 4カ国語サポート
韓国語、英語、日本語、中国語で提供され、世界中の開発者が活用できます。

## 対象読者

- **Claude Codeを始める開発者**: 基本概念から環境設定までPart 1から始めてください。
- **すでにClaude Codeを使用中の開発者**: Part 2-3で高度なテクニックを学んでください。
- **マルチエージェントシステムを構築したい開発者**: Part 3-4のアーキテクチャガイドを参照してください。
- **チームにAIツールを導入したいテックリード**: Part 5のエンタープライズガイドを確認してください。

---

## この本の読み方

```
推奨学習パス:

初心者: Part 1 → Part 2 → Part 3 (順次)

中級者: Part 2開始 → 必要なレシピ選択

上級者: Part 4から開始 → 関心のあるパターン深化

実践適用: Part 5のプロジェクトに沿って実践
```

各レシピは次の構造に従います:

| セクション | 説明 |
|------|------|
| **問題 (Problem)** | 解決しようとする具体的な問題 |
| **解決策 (Solution)** | 段階的解決方法 |
| **コード (Code)** | すぐに使用可能なコード例 |
| **説明 (Explanation)** | 動作原理と核心概念 |
| **変形 (Variations)** | 応用および拡張方法 |
