## 作者简介

作为一名开发者,我对利用AI工具提升生产力有着浓厚的兴趣。本书基于在实际博客运营项目中构建和运营11个AI Agent系统的经验,分享的是经过实战验证的知识,而非理论。

在撰写71篇博客文章的过程中积累的经验和教训都收录在这本书中。

---

## 本书特色

### 1. Cookbook风格
本书不是需要从头到尾顺序阅读的书。它设计为可以找到所需的配方(Recipe)并立即应用。

### 2. 实战验证的代码
所有示例都是项目中实际使用的代码。提供的不是理论性示例,而是可以直接复制使用的实用代码。

### 3. 4种语言支持
提供韩语、英语、日语、中文版本,全球开发者都可以使用。

## 目标读者

- **刚开始使用Claude Code的开发者**: 从Part 1的基本概念和环境设置开始。
- **已在使用Claude Code的开发者**: 在Part 2-3学习高级技巧。
- **想构建多Agent系统的开发者**: 参考Part 3-4的架构指南。
- **想在团队中引入AI工具的技术负责人**: 查看Part 5的企业指南。

## 如何阅读本书

```
推荐学习路径:

初学者: Part 1 → Part 2 → Part 3 (顺序学习)

中级者: 从Part 2开始 → 选择需要的配方

高级者: 直接从Part 4开始 → 深入感兴趣的模式

实战应用: 跟随Part 5的项目实践
```

每个配方遵循以下结构:

| 章节 | 说明 |
|------|------|
| **问题 (Problem)** | 要解决的具体问题 |
| **解决方案 (Solution)** | 分步解决方法 |
| **代码 (Code)** | 可直接使用的代码示例 |
| **说明 (Explanation)** | 工作原理和核心概念 |
| **变体 (Variations)** | 应用和扩展方法 |
